Presentación del tema

Presentación

Se plantean en este tema dos posibles casos prácticos de aplicaciones de datos LiDAR del IGN. Cada estudiante elegirá uno de ellos, del que entregará una práctica para su evaluación. En ambos casos se trabajará con el software ArcGIS.

Opción 1: Tratamiento y aplicaciones de datos LiDAR para estudios forestales y de vegetación.

1.1. Descarga archivos LiDAR del CNIG

1.2. Creación de un LAS Dataset 

1.3. Creación de MDE y MDS 

1.4. Estimación de la densidad y la altura de la cobertura forestal 

1.5. Creación de imágenes de intensidad

Opción 2: Tratamiento y aplicaciones de datos LiDAR para estudios urbanos de potencial solar.

2.1. Descarga archivos LiDAR del CNIG

2.2. Creación de un LAS Dataset 

2.3. Creación de MDE y MDS

2.4. Cálculo del potencial solar
